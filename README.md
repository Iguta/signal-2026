# signal-2026

A personal productivity app focused on goal and task tracking with a calm, modern UI.

## Overview
signal-2026 provides:
- Task management (add/edit/delete, descriptions, due dates, priorities, search)
- Goal setting (yearly, monthly, daily; thematic goals and custom categories)
- Calendar visualization with expand/zoom interactions
- Progress dashboards with daily/weekly/monthly/yearly insights

## UI/UX Principles
- Minimal, modern design with a default dark theme and soft accents
- Calendar-first layout and clear task visualization
- Responsive across mobile, tablet, and desktop

## Non-Functional
- No authentication required (personal use)
- Emphasis on clarity and calm
